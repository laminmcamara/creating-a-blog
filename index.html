What is Machine Learning?

My Viewpoint and Understanding!

Machine learning is an aspect of artificial intelligence where we train computers to perform tasks traditionally done by humans.

It enables computers to learn from data and improve their performance over time without being explicitly programmed.

The recent advancement of technology and Big Data is inevitably making machine learning significant for all industries, including healthcare, finance, and technology.

Types of Machine Learning

Machine learning can be categorized into several types but mainly three:

Supervised Learning:


In simple terms, when we are training a machine learning algorithm with a label dataset which enhances the algorithm to learn the patterns between the input and the outcomes, this type of training is called SUPERVISED learning.

If for example I want the machine to identify whether an email is 'spam' or 'not spam', I have to manually label the training data with the characteristics of 'spam' emails based on their history before passing it to the algorithm. By this way the machine learning model will learn from the pattern to give accurate prediction on a new data.


Another example of a supervised learning is training a deep learning model on image prediction. My dataset with pictures of different animals will be labelled with the exact names of these animals before passing to train the model, after this training the model will be able to predict the new data even without labelling them again and again. The more data you train the machine the more smarter it becomes.

Unsupervised Learning:

Unsupervised Learning is the direct opposite of supervised learnining where a machine learning model is train on a given data with no labels. Since it is learning without any labels, the model tries to identify patterns, groupings and the structure of the given data in order to be able to make predictions.

Characteristics of unsupervised learning:

Some key aspects of unsupervised learning include :

  1. the data given has no label to teach the model
  2. Clustering: this is where the model group together data points with similarities
  3. Dimensional reduction: here the model uses techniques like Principal Component Analysis PCA to reduce the data into it's most important components for easy analyis.
  4. Anomaly detection: this is where the model learns about the irregular patterns within the data also called Outliers. Transactions that are not normal will be identified just like fraud detection.

Common uses of unsupervised learning :

  • Customer segmentation
  • Market basket analysis
  • Credit card fraud detection
  • Document clustering
  • Image detection
  • Network security anomaly detection
